    Как опубликовать Сайт на github

    Лёгко!
    14/12/2021

    Для начала: что такое github?

    GitHub – один из крупнейших веб-сервисов для совместной разработки программного обеспечения .
    Есть бесплатные и платные тарифные планы пользования сайтом. Базируется на системе управления версиями Git
    и разработан на Ruby on Rails и Erlang компанией GitHub, Inc (ранее Logical Awesome).

    Итак... Как опубликовать Сайт на github

    Создаем новый акаунт githubа и новый репозиторий.



    Имя нашего репозитория называем "имя вашего акаунта.github.io"

    ВАЖНО!!! Обязательно пишем имя нашего акаунта.

    Понял?

    Дальше обязательно ставим галочку "Public"(Публично),
    но если вы хотите приватный репозиторий, то тогда с вас возьмут деньги
    ( но это не точно )

    И ещё ставим галочку "README".

    И жмём кнопку "Создать репозиторий"



    Дальше сюда загружаем файлы "Кнопка add file"


    Любуемся сваемь сайтикомь


    Просто жмёмь сюда




      Минусы и плюсы GitHub Pages



    Минусы:

    1. Ваш сайт всегда будет связан с GitHub,
    а это значит что если сайт GitHub вдруг упадёт,то и упадёт ваш сайт

    Других минусов я не вижу


    Плюсы:

    1. Вы можете полностью бесплатно создать сайт и
    загрузить его в сеть!

    2. Не нужно беспокоиться о сайте (сайт живёт своей жизнью)
